# Инструкция по подключению Яндекс.Карт к сайту Metro Park

## Что уже сделано в коде

В файле `index.html` уже вставлен скрипт подключения Яндекс.Карт:

```html
<script src="https://api-maps.yandex.ru/2.1/?apikey=YOUR_YANDEX_MAPS_API_KEY&lang=ru_RU" type="text/javascript"></script>
```

В файле `main.js` реализована инициализация карты с кастомной иконкой `map-pin-mp.svg`.

---

## Шаги для активации карты

### 1. Зарегистрировать API-ключ

1. Перейдите на [developer.tech.yandex.ru](https://developer.tech.yandex.ru/)
2. Авторизуйтесь через аккаунт Яндекс
3. Нажмите **«Подключить API»** → выберите **«JavaScript API и HTTP Геокодер»**
4. Заполните форму:
   - Название проекта: `Metro Park`
   - Домен сайта: укажите точный домен, где будет размещён сайт (например `metro-park.by`)
   - Если тестируете локально, добавьте `localhost`
5. Получите API-ключ вида `xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx`

> **Важно:** Бесплатный тариф — до 1000 запросов/сутки. Для сайта-лендинга этого достаточно.

---

### 2. Вставить ключ в HTML

Откройте `index.html`, найдите строку перед `</body>`:

```html
<script src="https://api-maps.yandex.ru/2.1/?apikey=YOUR_YANDEX_MAPS_API_KEY&lang=ru_RU" ...></script>
```

Замените `YOUR_YANDEX_MAPS_API_KEY` на ваш ключ:

```html
<script src="https://api-maps.yandex.ru/2.1/?apikey=ваш-ключ-здесь&lang=ru_RU" type="text/javascript"></script>
```

---

### 3. Проверить работу карты

После замены ключа и публикации сайта:
- Откройте страницу → прокрутите до раздела «Контакты»
- На карте должна появиться золотая иконка Metro Park (MP) с адресом ул. Космонавтов, 10А
- Кнопки «Доехать» открывают Яндекс.Карты и Google Maps с уже вбитым адресом

---

## Иконка для карты (map-pin-mp.svg)

Файл `map-pin-mp.svg` уже создан и находится рядом с `index.html`. Это SVG-иконка в виде перевёрнутой капли с текстом **MP** в цвете логотипа (`#c4a882`).

При деплое сайта убедитесь, что `map-pin-mp.svg` загружен на сервер в той же папке, что и `index.html`.

## Стилизация карты

Карта в v4 работает **без CSS filter** (убран `filter: invert(...)` из предыдущей версии). Стандартная карта Яндекса подходит к тёмному сайту за счёт тонкой рамки и нейтрального фона.

Если нужна более тёмная карта — Яндекс.Карты 2.1 не имеют нативной «тёмной темы» как Google Maps. Альтернативы:
1. **Спутниковый тип карты** — тёмный по умолчанию:
   ```js
   type: 'yandex#satellite'
   ```
2. **Пользовательский стиль через JSON** — требует тарифа «Коммерческий»
3. **Оставить стандартную карту** — наиболее совместимый вариант (текущая реализация)

## Кнопки маршрута

Обе кнопки уже настроены с адресом назначения:

- **Яндекс.Карты:** открывает маршрут с точкой «Куда» = г. Витебск, ул. Космонавтов, 10А
- **Google Maps:** открывает Google Maps Directions с тем же адресом

---

## Тестирование без ключа

Без ключа карта будет показывать сообщение «Недействительный API-ключ». Это нормально при разработке. После вставки реального ключа карта загрузится.
